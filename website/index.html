Designed for developers

Gooey provides everything you need to create beautiful, native-feeling applications across platforms.

Cross-Platform

Write once, build anywhere. Gooey applications work seamlessly on Windows, and Linux with native performance.

Lightweight

Minimal footprint with maximum functionality. Gooey is designed to be fast and efficient without bloat.

Customizable

Tailor every aspect of your UI. From themes to custom widgets, Gooey gives you complete control.

Minimal Dependencies

Built with minimal dependencies.

Built in pure C

Portable code. Perfect for Embedded Solutions and hardware with limited resources.
